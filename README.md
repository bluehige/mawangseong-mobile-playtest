# 마왕성 모바일 플레이테스트

스마트폰과 태블릿의 가로 화면을 위한 공개 모바일 브라우저 테스트 빌드입니다.

- 플레이: https://bluehige.github.io/mawangseong-mobile-playtest/
- 기준 소스: `bluehige/mawangseong-demo@bb61ba99a9403532345d441233a8c59821b3fecd`
- 빌드 엔진: Godot 4.5.2
- 권장 환경: 최신 Chrome 또는 Safari, 가로 화면
- 포함 변경: Lyria BGM·스킬별 효과음, 큰 글꼴·터치 버튼, 강조 링 전체 탭, 튜토리얼 사전 선택, 첫 터치 가로 전체화면, 가상 키보드 방해 제거, 모바일 전용 경량 렌더·CSS 픽셀 백버퍼·Web BGM 스트리밍

- 모바일 PCK: 146,803,604바이트. Noto 폰트·전투 스프라이트·오디오는 원본 품질을 유지하고 큰 일러스트만 품질 0.90·최대 1280px로 최적화했습니다.

네이티브 APK/AAB가 아닌 모바일 브라우저용 테스트 빌드입니다.
